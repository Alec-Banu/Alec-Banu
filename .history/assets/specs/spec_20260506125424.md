# Home Page Spec — Alec Mitroi Banu

## Overview

A personal portfolio home page introducing who I am, my skills, and my projects.
Target audience: potential employers and collaborators.

## Sections

- **Nav** — sticky header with anchor links to each section
- **Hero** — full-viewport intro with name, tagline, and CTA button
- **About** — short bio paragraph
- **Skills** — tag-style pills listing: HTML, CSS, JavaScript, C#, PowerShell
- **Projects** — card grid (3 cards) with title and description
- **Contact** — short message with mailto link styled as a button
- **Footer** — credit line

## Design Decisions

- Color scheme: dark navy background (`#0a192f`), light slate text (`#ccd6f6`), teal accents (`#64ffda`)
- Mobile-first layout; nav centers and wraps on small screens, right-aligns at 768px+
- Project cards stack on mobile, 2 columns at tablet, 3 columns at desktop
- Hero takes full viewport height with vertically centered content
- Fonts and base rules in `globals.css`; page-specific styles in `index.css`

## Known Limitations

- Project cards contain placeholder content (no real projects yet)
- No images currently on the page
